# 💎 ProphetReal AI: Global Property Intelligence (World Edition v3.5)

![Project Aesthetic](https://img.shields.io/badge/Aesthetic-Obsidian%20%26%20Gold-d4af37?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Tech-XGBoost%20%7C%20FastAPI%20%7C%20Streamlit-000000?style=for-the-badge&logo=python)

**ProphetReal AI** is a state-of-the-art property valuation ecosystem engineered for the modern real estate landscape. By unifying advanced Machine Learning with high-performance web architecture, ProphetReal provides hyper-accurate, real-time valuations across both **Indian (INR)** and **Global (USD)** markets.

---

## ⚡ Key Intelligence Features

- **🎯 Dual-Market Predictive Engine**: Leveraging high-fidelity XGBoost regressors trained on localized datasets. From BHK and Metro Proximity in India to Overall Quality and Smart Home metrics globally.
- **⚖️ Strategic ROI Simulator**: A first-of-its-kind "Market Sentinel" tool. Simulate property upgrades—like smart home integration or premium security—and instantly visualize the projected equity growth.
- **📊 Sentinel AI Insights**: Beyond simple price prediction, the system provides strategic advisory on infrastructure premiums, depreciation offsets, and investment viability.
- **🎨 Elite Obsidian Interface**: A premium dashboard designed with a dark Obsidian aesthetic and Gold accents, featuring interactive Plotly visualizations for data-driven storytelling.

---

## 🛠️ The Tech Stack behind the Intelligence

- **Machine Learning**: XGBoost Regressors (Custom tuned for dual-market volatility).
- **Core Architecture**: FastAPI for sub-millisecond inference and Pydantic for strict data integrity.
- **Dashboard**: Streamlit + Plotly (Interactive gauges, feature importance bar charts, and glassmorphism UI).
- **Data Engineering**: Specialized synthetic generation pipelines for Indian and Global market contexts.

---

## 🚀 Market Versatility

### 🇮🇳 Indian Market Engine
Optimized for the Indian context, supporting:
- **City Tiers**: Metro, Tier 2, and Tier 3 localization.
- **Infrastructure**: Power backup, Gated Security, and Metro Proximity.
- **Valuation**: Smart currency formatting in Cr/Lakhs.

### 🌎 Global Market Engine
Built on Ames-inspired standards, focusing on:
- **Quality metrics**: Overall Condition, Kitchen Quality, and Eco-Certification.
- **Modern Amenities**: Smart Home ecosystems and Basement utility.
